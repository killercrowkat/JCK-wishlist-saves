# JCK Wishlist Saves

This repository serves as the public data backbone for the [JCK Wishlist Site](https://jck-wishlist-site.vercel.app/). It hosts JSON files containing curated wishlists that anyone can subscribe to.

You can use the files in this repository to auto-populate your own wishlist without manual typing.

### Step-by-Step Guide:
1.  **Find a list:** Browse the `.json` files in this repository (e.g., `JCK-MAIN-WISHLIST.json`).
2.  **Get the Raw Link:** Click on the file, then click the **"Raw"** button at the top right of the file view.
3.  **Copy the URL:** It should look like this:
    `https://raw.githubusercontent.com/killercrowkat/JCK-wishlist-saves/refs/heads/main/your-file.json`
4.  **Sync to the App:** * Open [JCK Wishlist](https://jck-wishlist-site.vercel.app/).
    * Find the **URL Source** input.
    * Paste the link and save.
